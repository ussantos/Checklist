\# AGENTS.md — Checklist



\## Contexto do projeto



Este é um sistema web interno chamado Checklist, usado pela My Robot Barra da Tijuca para acompanhar rotinas operacionais de colaboradores.



A aplicação é Django + PostgreSQL + Docker Compose.



O sistema deve ser simples, interno, auditável e seguro. Não deve ser exposto diretamente à internet.



\## Stack



\- Python 3.12

\- Django

\- PostgreSQL

\- Docker Compose

\- Gunicorn

\- WhiteNoise
